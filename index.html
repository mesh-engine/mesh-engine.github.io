Game Engine For Play-to-Earn

Mesh provides you set of tools to build your own play-to-earn game.


Mesh is a 2D and 3D game engine that is powered by set of smart contracts to create play-to-earn games targeting PC, mobile, and web platforms.

A different way to make games

Powerful 3D

Innovative 3D renderer design, which makes your art look great with minimal effort.

Beautiful 2D

Dedicated 2D engine that works in pixel coordinates, with plenty of built-in tools.

Unique Architecture

Big or small ideas adapt seamlessly to Mesh node-based architecture, making your life easier.

Easy to use

Object-oriented API with language options such as C#, C++ and visual scripting.

Integrated Smart Contracts

Use Ethereum or Binance Smart Chain smart contracts just as using a database to store immutable data. Use Mesh play-to-earn gaming specific smart contracts solve your payment related needs.

Integrated Subgraphs

Query Ethereum or Binance Smart Chain using Mesh play-to-earn gaming specific subgraphs. Seamlessly attach your query data to Mesh Engine assets to create fully integrated workflows.

Meet MESH Token

MESH Token is the native token of Mesh Engine that will be used in up coming MESH store that sells gaming related 3D files, engine addons and smart contracts codes. MESH is a hyper-deflationary token with an integrated smart staking system, so you're being rewarded automatically with more MESH straight to your wallet for simply holding. Ownership is renounced and the liquidity of token is locked and cannot be accesed by anyone. To reward holders of the token we redistribute 4% of every transaction to all holders.
